## ✨ About the Project | درباره پروژه

Proxy Pro Advanced is a fast and configurable command-line tool designed to collect, test, filter and manage proxy servers.

Proxy Pro Advanced یک ابزار سریع و قابل تنظیم برای جمع‌آوری، تست، فیلتر و مدیریت سرورهای پروکسی است.

### Main capabilities | امکانات اصلی

- Collect proxies from public and private sources
- Test proxy availability and connection speed
- Support for HTTP, HTTPS, SOCKS4 and SOCKS5
- Detect country, ISP and anonymity level
- Separate working, failed and high-quality proxies
- Export results in TXT, CSV and HTML formats
- Simple installation and execution on Windows
- Persian and English documentation

## 🚀 Quick Start | شروع سریع

1. Download the latest release.
2. Extract the ZIP file.
3. Run INSTALL_FIRST.bat for the first installation.
4. Run RUN.bat to launch the tool.

<br>

1. آخرین نسخه را دانلود کنید.
2. فایل ZIP را استخراج کنید.
3. برای نصب اولیه، فایل INSTALL_FIRST.bat را اجرا کنید.
4. برای اجرای ابزار، فایل RUN.bat را اجرا کنید.
